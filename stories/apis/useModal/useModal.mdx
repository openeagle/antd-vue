# Modal 对话框

模态对话框。打开一个浮层显示相应内容

另外当需要一个简洁的确认框询问用户时，可以使用 `Modal.confirm()` 等语法糖方法。

## 代码演示

**导入hook**

```jsx
import { useModal } from '@openeagle/antd-vue';
```

#### 属性

<div className="mdTable">


| 参数              | 说明                                                         | 类型                                                     | 默认值    |
| :---------------- | :----------------------------------------------------------- | :------------------------------------------------------- | :-------- |
| autoFocusButton   | 指定自动获得焦点的按钮                                       | String `ok` `cancel`                                     |           |
| centered          | 垂直居中展示 Modal                                           | Boolean                                                  | `true`    |
| closable          | 是否显示右上角的关闭按钮                                     | boolean                                                  | `false`   |
| class             | 容器类名                                                     | string                                                   |           |
| content           | 内容                                                         | string                                                   | vNode     |
| icon              | 自定义图标                                                   | VNode                                                    | ()=>VNode |
| mask              | 是否展示遮罩                                                 | Boolean                                                  | true      |
| maskClosable      | 点击蒙层是否允许关闭                                         | Boolean                                                  | `false`   |
| keyboard          | 是否支持键盘 esc 关闭                                        | boolean                                                  | true      |
| footer            | 底部内容，当不需要默认底部按钮时，可以设为 `:footer="null"`  | string                                                   | slot      |
| footerPosition    | 底部内容位置                                                 | string  left                                             | right     |
| cancelText        | 取消按钮文字                                                 | string                                                   | 取消      |
| cancelType        | 取消按钮类型                                                 | string                                                   | default   |
| okText            | 确认按钮文字                                                 | string                                                   | 确定      |
| okType            | 确认按钮类型                                                 | string                                                   | primary   |
| okButtonProps     | ok 按钮 props                                                | [ButtonProps](https://2x.antdv.com/components/button-cn) |           |
| cancelButtonProps | cancel 按钮 props                                            | [ButtonProps](https://2x.antdv.com/components/button-cn) |           |
| btns              | 自定义显示按钮                                               | [btnType](#btntype对象)[]                                |           |
| title             | 标题                                                         | string                                                   | vNode     |
| width             | 宽度                                                         | string                                                   | number    |
| zIndex            | 设置 Modal 的 `z-index`                                      | Number                                                   | 1000      |
| onCancel          | 取消回调，参数为关闭函数，返回 promise 时 resolve 后自动关闭 | function                                                 |           |
| onOk              | 点击确定回调，参数为关闭函数，返回 promise 时 resolve 后自动关闭 | function                                                 |           |
| parentContext     | 弹窗的父级上下文，一般用于获取父级 provider， 如获取 `ConfigProvider` 的配置 | vue instance                                             |           |
| autoDestroy       | 离开时是否自动销毁                                           | Boolean                                                  | true      |

</div>




#### btnType对象

```
btnType {
  btnText: VNodeTypes; // 按钮文字
  btnType?: btnType; // 按钮类型 'default' | 'primary' | 'ghost' | 'dashed' | 'danger' | 'link'
  onClick?: () => any; // 按钮事件
  props?: any; // 按钮 props
}
```

## SearchSelect 远程搜索

通过搜索实时从后台获取数据，并下拉选择。

### 何时使用

数据量大的情况下，不想一次全部获取，可以使用该组件。

### 代码演示

**导入组件**

```jsx
import { SearchSelect } from '@openeagle/antd-vue';

```

<Story id="组件-numbertext--api" />

### API

> 本组件是对 antd vue 中的 Select 选择器的二次封装，所以支持 Select 所提供的所有属性，可以参见官网：https://2x.antdv.com/components/select-cn#API
> 下面只对自定义的一些属性进行说明。


#### 属性




<div className="mdTable">

>

| 参数           | 说明                     | 类型              | 可选值 | 默认值 |
| -------------- | ---------------------------------------- | ----------------- | ------ | ------ |
| value(v-model) | 	指定当前选中的条目       | string|string[]|number|number[]  |        |     |
| allowEmptySearch | 是否允许空值查询       | Boolean             |        |     |
| mode      | 设置 Select 的模式为多选或标签     | String            |  	'multiple' | 'tags' | 'combobox'      |     |
| dataSource     | 必填，远程数据源            | Function as PropType<(keyword: string) => Promise<any[]>>  |        |      |
| extras          | 额外的数据，就是都会在列表显示的数据，具体用法看上面 `extras` Demo   | Array as PropType<any[]>             |        |     |
| fieldNames |    选项字段映射，具体用法看上面 `preset` Demo      |  Array as PropType<any[]>  |        |   `{ label: 'label', value: 'value' }`  |

</div>

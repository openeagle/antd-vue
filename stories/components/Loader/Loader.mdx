## Loader 请求状态处理

主要针对请求失败做错误处理


<Story id="组件-loader--api" />

### API

<div className="mdTable">

| 参数    | 说明               | 类型                                              | 可选值                                     | 默认值          |
| :------ | ------------------ | ------------------------------------------------- | ------------------------------------------ | --------------- |
| failure | 支持v-model        | [failure](#failure对象)                           | 必传                                       |                 |
| loading | 上传列表的内建样式 | Boolean \| {     size: String    tip: String;   } | {  size: 'small' \|'default' \|'large'   } | {size: 'large'} |
| success | 支持按文件夹上传   | [success](#success对象)                           |                                            | false           |

</div>



### failure对象
```
failure: {
  error?: Error | EaxiosError;
  okText?: string;
  cancelText?: string;
  onOk?(): any;
  onCancel?(): any;
  options?: {
    [key: string]: {
      status:
        | 'success'
        | 'error'
        | 'info'
        | 'warning'
        | '404'
        | '403'
        | '500';
      title?: string;
      subTitle?: string;
      okText?: string;
      cancelText?: string;
      onOk?(): any;
      onCancel?(): any;
    };
  };
}

EaxiosError<T = any> extends Error {
  config: EaxiosRequestConfig;
  code?: string;
  request?: any;
  response?: EaxiosResponse<T>;
  isAxiosError: boolean;
  toJSON: () => object;
}
```



### success对象
```
success?: {
  status: 'success' | 'error';
  icon: VNode;
  title: string | VNode;
  subTitle: string;
  extra: VNode;
};
```

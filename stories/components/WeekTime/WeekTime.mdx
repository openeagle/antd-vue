## WeekTime 时间段选择器

将时间以半小时为单位，分割成 48 个时间片，用于需要时间管理的人。

### 何时使用

需要选择多个时间段的时候使用。

### 代码演示

**导入组件**

```jsx
import { WeekTime } from '@openeagle/antd-vue';
import weekTimeData, {
  splicing,
} from '@openeagle/antd-vue/es/components/WeekTime/weektimeData';
```

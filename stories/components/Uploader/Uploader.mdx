## Upload 上传

文件选择上传和拖拽上传控件。

上传是将信息（网页、文字、图片、视频等）通过网页或者上传工具发布到远程服务器上的过程。



<Story id="组件-uploader--api" />

### API

下面只说明比较常用和自己封装的属性，其它具体配置可以前往 https://2x.antdv.com/components/upload-cn 进行学习查看。

#### 属性


<div className="mdTable">

| 参数            | 说明                                                         | 类型                      | 可选值                       | 默认值 |
| :-------------- | ------------------------------------------------------------ | ------------------------- | ---------------------------- | ------ |
| file-list       | 支持v-model                                                  | Array                     |                              |        |
| listType        | 上传列表的内建样式                                           | String                    | text,、picture、picture-card | 'text' |
| directory       | 支持按文件夹上传                                             | Boolean                   |                              | false  |
| disabled        | 是否禁用(无法上传)                                           | Boolean                   |                              | false  |
| action          | 上传的地址                                                   | String\|(file) => Promise |                              | 1      |
| multiple        | 是否支持多选文件，`ie10+` 支持。开启后按住 ctrl 可选择多个文件。 | Boolean                   |                              | false  |
| name            | 发到后台的文件参数名                                         | String                    |                              | 'file' |
| withCredentials | 上传请求时是否携带 cookie                                    | Boolean                   |                              | false  |

</div>


#### 事件


<div className="mdTable">

| 事件名称           | 说明              | 回调参数 |
| -------------- | ---------------------------------------- | ----------------- |
| change | 上传文件改变时的状态，详见 [ant change](https://2x.antdv.com/components/upload-cn#change) | Function |
| preview | 点击文件链接或预览图标时的回调 | Function(file) |
| download | 点击下载文件时的回调，如果没有指定，则默认跳转到文件 url 对应的标签页。 | Function(file): void |
| reject | 拖拽文件不符合 accept 类型时的回调 | Function(fileList) |

</div>

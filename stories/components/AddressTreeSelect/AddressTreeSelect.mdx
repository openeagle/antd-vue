## AddressTreeSelect 地址选择器

用于地址选择。

### 引入

```jsx
import { AddressTreeSelect } from '@openeagle/antd-vue';
```

### API


#### 属性


<div className="mdTable">

| 参数           | 说明 | 类型              | 可选值 | 默认值 |
| -------------- | ---------------------------------------- | ----------------- | ------ | ------ |
| selects (v-model) | 指定选中项                 |      string[]             |        |      |
| selectNames (v-model) | 指定选中项 label 值    |      string[]             |        |      |
| title        |  显示的标题         |   string[]      |        | 'area'     |    []
| data          |  可选项数据源         |  Option[]     |        |    |
| loading          |  设置载入状态         |  boolean     |        | false   |
| fieldNames        |  自定义 options 中 label name children 的字段         |   	object     |        |   { value: 'code' }   |

</div>


### Option

```
interface Option {
  "id": 3000,
  "key": "tj",
  "value": "天津"
}
```

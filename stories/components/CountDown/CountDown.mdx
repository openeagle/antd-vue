## CountDown 倒计时

用于实时展示倒计时数值，支持毫秒精度。

<div className="storybook-note">
  本组件是移动于 Vant 中的 count-down 组件，详细 API 可以查看对应的官网：
  <br />
  <a href="https://vant-contrib.gitee.io/vant/v3/#/zh-CN/count-down#shou-dong-kong-zhi">
    https://vant-contrib.gitee.io/vant/v3/#/zh-CN/count-down#shou-dong-kong-zhi
  </a>
</div>

#### 引入

```jsx
import { CountDown } from '@openeagle/antd-vue';
```
